# Project info

Framework: Vue3
State management: Pinia
Http library: Axios
Chart library: chart.js
Unit tests: Jest
Development Server: Vite

Css pattern: Css modules
Css methodology: BEM

## Project setup

Go to the root path of weather-project and run

```
npm install
```

### Compile and run the project

```
npm run dev
```

### Run Jest unit tests

```
npm run test:unit
```
